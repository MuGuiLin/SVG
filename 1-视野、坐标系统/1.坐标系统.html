    SVG标签属性：

    <svg xmlns="http://www.w3.org/2000/svg" width="800" height="600" viewBox="0 0 400 300" preserveAspectRatio="xMidYMid meet" >

  • width="800" height="600" // 控制视窗宽高大小【注：不指定宽高时：默认为300 * 150】
  • viewBox="x y w h" // 控制伸展位置、宽高大小
  • preserveAspectRatio="xMidYMid meet" // 控制视野缩放、显示方式
