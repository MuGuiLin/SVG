什么是SVG

SVG 是由万维网联盟（W3C[World Wide Web Consortium]）自 1999 年开始开发的开放标准。

SVG 是一种基于XML（Extensible Markup Language）的，描述二维的，矢量图形语言。

SVG是一种图像文件格式，它的英文全称为可缩放矢量图形（Scalable Vector Graphics，SVG），意思为可缩放的矢量图形。可让你设计激动人心的、高分辨率的Web图形页面。 用户可以直接用代码来描绘图像，可以用任何文字处理工具打开SVG图像，通过改变部分代码来使图像具有交互功能，并可以随时插入到HTML中通过浏览器来观看。

SVG能够优雅而简洁地渲染不同大小的图形，并和CSS，DOM，JavaScript和SMIL等其他网络标准无缝衔接。本质上，SVG 相对于图像，就好比 HTML 相对于文本。

SVG 图像及其相关行为被定义于  文本文件之中，这意味着可以对它们进行搜索、索引、编写脚本以及压缩。此外，这也意味着可以使用任何文本编辑器和绘图软件来创建和编辑它们。

和传统的点阵图像模式，像JPEG和PNG不同，SVG格式提供的是矢量图，这意味着它的图像能够被无限放大而不失真或降低质量，并且可以方便地修改内容。

  • 1、SVG 指可伸缩矢量图形 (Scalable Vector Graphics)；
  • 2、SVG 用来定义用于网络的基于矢量的图形；
  • 3、SVG 使用 XML 格式定义图形；
  • 4、SVG 图像在放大或改变尺寸的情况下其图形质量不会有所损失；
  • 5、SVG 是万维网联盟的标准；
  • 6、SVG 与诸如 DOM 和 XSL 之类的 W3C 标准是一个整体 ；
